#  Data Science Project 1 — E-Commerce Sales Analysis & Demand Prediction

## Objective
Analyze past e-commerce sales data and build a machine learning model to predict 
future product demand, helping businesses optimize inventory management.

##  Dataset
- **Source:** Kaggle — E-Commerce Sales Dataset
- **Size:** 34,500 rows × 17 columns
- **Features:** Order ID, Product Category, Price, Discount, Quantity, Region, 
  Profit Margin, Payment Method, Delivery Time, Customer Demographics, and more.

##  Tools & Libraries
- **Data Handling:** Python, Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
- **Machine Learning:** Scikit-learn (Linear Regression, Random Forest Regressor)

##  Project Workflow
1. Install & Import Libraries
2. Load & Clean Dataset
3. Exploratory Data Analysis (EDA)
4. Machine Learning — Demand Prediction
5. Business Insights & Conclusion

## Key Analyses
- Monthly sales trend over time
- Top 10 products by revenue
- Region-wise sales performance
- Seasonal sales patterns
- Category-wise sales distribution

## ML Models Used
| Model | Description |
|-------|-------------|
| Linear Regression | Baseline model |
| Random Forest Regressor | Ensemble model (better accuracy) |

**Target Variable:** `quantity` (units demanded per order)
